手机软件推荐

  • 下面👇将分：文件管理📁，音乐软件🎵，信息获取软件📄，笔记软件📝，学习软件🤓分栏介绍。

文件管理📁

MT管理器

  • 功能：双窗口文件管理、APK逆向修改、强大文本编辑、压缩包处理、脚本执行

  • 介绍：一款在Android平台上集专业文件管理与深度APK编辑于一体的强大工具。它以独特的左右双窗口操作和无需解压直接编辑压缩包/APK文件的能力著称,其部分高级功能需要VIP会员才能使用。

  • 适配平台：
    Android (要求 Android 4.2 以上)

  • 官方网站 (https://mt2.cn)

NP管理器

  • MT管理器的平替与它的功能基本相同👆
  • 免费
  • 适配平台：Android
  • 官方网站（http://normalplayer.top）

ZArchiver

  • 功能：全格式压缩/解压、压缩包内文件预览、加密压缩、分卷压缩处理
  • 介绍：一款用于管理压缩文件的免费应用程序。它拥有简洁而功能丰富的界面，支持处理多种压缩格式，包括 7zip、zip、bz2、gz2、tar 等。
  • 适配平台：Android
  • 官方网站：（https://www.zdevs.ru/）

音乐软件🎵

Salt Player(椒盐音乐)（安卓最好的音乐播放器）

  • 功能：全格式音频播放（含无损格式）、本地音乐管理与分类、专业音质调节（均衡器 / DSP 音效）、播放模式切换（随机 / 循环 / 单曲）、动态流光界面、多屏模式适配（横屏 / 沉浸）
  • 介绍：一款免费无广告的本地音乐播放器，专注纯净听歌体验。支持 MP3、FLAC、APE、DSD 等多格式音频，配备专业音质引擎与动态界面效果，功能全面且界面简洁无干扰。
    这是在安卓可以找到的最优秀，最美观的本地音乐播放器
  • 适配平台：安卓
  • 官方网站（https://moriafly.com/program/salt-player.html）
  • github仓库：（https://github.com/Moriafly/SaltPlayerSource）或各大应用市场

VLC（比音乐软件要牛逼👍）

  • 功能：全格式音视频播放（覆盖 MP4/AVI/FLAC/DSD 等格式，无需额外解码器）、流媒体播放 / 录制（网络串流、直播源播放 + 实时录制）、音视频转码（格式转换、编码 / 参数自定义）、字幕同步（本地 / 在线字幕匹配、样式 / 时间轴调整）、播放增强（倍速 / 投屏 / 音频滤镜 / 视频特效）
  • 介绍：一款开源免费的跨平台多媒体工具，以 “全格式兼容 + 无解码器依赖” 为核心优势，无广告且功能覆盖播放、录制、转码等多场景，兼顾普通用户的便捷性与进阶用户的专业需求。
  • 适配平台：Android（手机 / 平板 / TV）、iOS、Windows、macOS、Linux、ChromeOS、智能电视（LG webOS/Samsung Tizen）
  • 官方网站：（https://www.videolan.org/vlc/）

MusicFree（壳子软件，要插件）

  • 功能：聚合多平台音乐源搜索 / 播放、歌词同步（实时滚动歌词、多源匹配 + 翻译功能）、歌单管理（自建 / 导入 / 云端同步歌单）、音频离线缓存、界面自定义（主题切换、布局调整）、插件扩展（音效插件、额外音源插件）
  • 介绍：一款免费开源的音乐聚合播放器，核心亮点是 “跨平台音源整合 + 无广告纯净体验”，无需切换多个音乐 APP 即可获取多平台音乐资源，支持无损播放、歌词同步等基础功能，同时提供插件扩展能力满足个性化需求，界面轻量简洁无冗余。
    是个壳子，需要插件，可以去QQ频道看看
  • 适配平台：Android、Windows
  • 官方网站：（https://musicfree.vercel.app）
    GitHub 仓库：（https://github.com/maotoumao/MusicFree）

信息获取软件📄

Capy Reader

  • 功能：多源订阅管理 (Feedbin/FreshRSS/Miniflux/ 本地源)、文章搜索与标记、大屏适配、订阅源导入导出、多账户管理、阅读进度跟踪
  • 介绍：一款轻量级免费 RSS 阅读器，以极简设计为核心，专注提供无干扰的内容阅读体验。支持多种 RSS 服务与本地订阅源，可快速解析并展示文章完整内容，界面简洁高效，适合内容爱好者和信息聚合需求用户。需要rss订阅源
  • 适配平台：Android 11 及以上版本 (手机 / 平板)
  • 官方网站：(https://capyreader.com/），GitHub仓库：(https://github.com/jocmp/capyreader)

Folo

  • 功能：全平台内容聚合 (RSS / 社交媒体 / 新闻网站)、AI 智能翻译与摘要、多类型内容支持 (文章 / 视频 / 图片 / 音频)、个性化信息中心、社区分享与探索、上下文 AI 助手 (背景补充 / 相关性分析)、动态内容解析、自定义分类与过滤、无干扰阅读体验、多设备同步

  • 介绍：一款免费开源的 AI 驱动信息聚合工具，专注解决信息过载问题。它能将分散在互联网各平台的内容 (包括 Twitter、微博、B 站等 1200 + 网站) 整合到单一时间线中，通过 AI 技术提供自动摘要、实时翻译和内容洞察，让用户高效获取有价值信息，同时保持界面简洁无干扰，完全摆脱 “信息茧房” 和 “未读焦虑。

  • 适配平台：全平台支持，包括 Web (网页版)/iOS/Android/macOS/Windows/Linux

  • 官方网站：(https://app.folo.is/)，GitHub 仓库：(https://github.com/RSSNext/Folo)

Reqable

  • 功能：
    • 全协议网络抓包分析 (HTTP/HTTPS/WebSocket/HTTP3)
    • API 测试与调试 (请求编辑 / 响应分析 / 断言验证)
    • 移动端独立抓包 (Android/iOS 无需 Root)
    • 流量重写 / 断点调试 / 脚本扩展
    • 多设备协同模式 (手机 <–> 电脑双向数据转发)
    • 历史记录与数据分析
    • 支持 Postman 等格式数据导入
  • 介绍：一款跨平台专业 HTTP 开发调试工具，被誉为 “Fiddler+Charles+Postman” 的全能集成方案。提供两大核心功能：抓包调试(实时监控应用网络请求) 和API 测试(设计 / 执行 / 管理 REST 接口测试)，并打通两者壁垒 (可直接从抓包数据生成 API 测试用例)。界面简洁、性能高效，支持全平台使用且无广告，是开发者、测试人员和网络安全爱好者的强大助手。
  • 适配平台：Windows/macOS/Linux/Android/iOS (全平台覆盖)
  • 官方网站：(https://reqable.com/) GitHub 仓库：(https://github.com/reqable/reqable-app)
  • 注：Reqable 是完全免费的开源工具 (部分高级功能需要VIP会员)，无强制登录，专注提供纯净的开发调试体验。

笔记软件📝

知拾笔记（原知识库）

  • 功能：多类型笔记（文本 / Markdown / 思维导图 / 网页剪藏）；多端同步（WebDAV / 阿里云盘备份、本地备份）；智能分类（分类 + 标签 + 树状图管理）；代码高亮、随机阅读、时间线视图；支持导出 PDF / 图片 / TXT，支持网页正文转 Markdown；隐私保护（指纹 / 手势加密、保密柜）。
  • 介绍：免费（部分高级功能需要VIP会员）开源的轻量知识管理工具，主打 “一站式知识收集 - 整理 - 输出”，适合构建个人知识体系，界面简洁、无广告，对程序员与内容创作者友好。
  • 适配平台：Android/Windows
  • 下载平台：Gitee（https://gitee.com/knowshub/knows-note/releases）

纯纯写作（Pure Writer）

  • 功能：自动保存 + 历史记录（时光机）双重防丢失；Markdown 原生支持与语法高亮；无限字数长文编辑，丝滑输入体验；多端同步（局域网桌面版协同、WebDAV / 阿里云盘 / OneDrive 云同步）；写作辅助（TTS 语音朗读、快捷输入栏、自动补全成对符号）；排版优化（行 / 段间距调整、中英文自动空格）；内容导出（PDF / 长图 / 短图 / 纯文本）；隐私保护（指纹 / 手势加密）；书籍 / 章节结构化管理、标签分类、字数统计。

  • 介绍：免费（部分高级功能需要VIP会员）无广告的轻量级文本编辑器，核心亮点是 “绝不丢失内容” 与 “极简流畅体验”。启动极速、界面简洁，无冗余干扰，兼顾基础编辑与进阶写作需求，适配长文创作、日记记录、技术文档编写等场景，对创作者友好且保障内容安全。

  • 适配平台：Android/Windows

  • 官方网站：（https://writer.drakeet.com/）

Joplin

  • 功能：Markdown / 富文本双编辑、多媒体（图片 / 视频 / PDF / 音频）嵌入、OCR 图片识别、数学公式与绘图创建；端到端加密（AES-256）、离线优先存储、笔记历史版本与回收站；笔记本 + 子笔记本层级管理、标签分类、全文搜索；待办事项（提醒 / 截止日期设置）；多渠道同步；网页剪藏；插件扩展、自定义主题、快捷键配置。

  • 介绍：免费 开源的跨平台笔记与待办事项工具，被誉为 “印象笔记的理想开源替代品”。以隐私保护为核心，开源透明且数据采用开放格式，无厂商锁定风险；端到端加密确保云端同步数据安全，离线优先设计保障无网络时也能正常使用。兼顾普通用户的操作便捷性与进阶用户的定制需求，适配个人记录、知识管理、项目协作等多场景，是重视数据主权用户的优选工具。

  • 适配平台：Windows/macOS/Linux/Android/iOS

  • 官方网站：（https://joplinapp.org/）
    Github仓库：（https://github.com/laurent22/joplin）

学习软件🤓

多邻国（Duolingo）

  • 功能：40 + 种语言课程学习（含主流语言、小众 / 濒危语言及人造语言），新增数学、音乐课程；游戏化闯关学习（积分、红心、王冠奖励机制），听说读写全维度练习；多设备数据同步；多邻国英语测试（DET）认证；家长端学习报告查看（适配儿童学习）。

  • 介绍：全球超 5 亿用户选择的免费教育类 APP，以 “游戏化碎片化学习” 为核心亮点，让语言学习摆脱枯燥。课程由语言专家设计，采用科学教学方法论，15 分钟 / 天即可循序渐进提升能力，核心课程完全免费，支持免费增值模式（Super/Duolingo Max 会员解锁免广告、无限红心、高级 AI 功能等）。适配零基础入门、日常交流、兴趣学习等场景，兼顾成人与儿童学习需求，是全球备受认可的语言学习工具。

  • 适配平台：Android、iOS、Windows、macOS

  • 官方网站：（https://www.duolingo.com/）
